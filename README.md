# lottoGameProject : Lotto 6/45 로또 시뮬레이션
### 로또는 많은 사람들이 즐기는 게임중 하나입니다. 시뮬레이션을 만들어 특정 번호의 빈도 수를 살펴보는 등 데이터 분석으로 사용할 수 있음

### ◆ 개발 환경
* Java 8
* JDK 1.8.0
* Eclipse

### ◆ 학습 목표
* 클래스, 객체에 대한 이해
* 배열과 랜덤 함수 사용법에 대한 이해
* 메서드의 작성과 활용
* Eclipse 숙련도

### ◆ 담당 업무
* 번호생성 / 수동, 자동 번호 입력 / 당첨 번호의 비교 / 통계 분석
* Lotto 규칙(룰) 숙지
* UI 구성 목표 계획
